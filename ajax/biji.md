code runner 代码执行器

mac 电脑上的 git 位置：
$ git --version ---查看是否安装 git
$ which git ---查看位置 /usr/local/bin/git

用终端管理 git 代码：
$ git add ---提交代码（把代码放在缓存区）
$ git commit -m '修改信息' ---修改了什么
$ git push origin master ---推送到仓库

$ git pull origin master ---拉取代码
